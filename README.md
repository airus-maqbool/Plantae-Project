# 🌿 Plantae

A cross-platform mobile app where users can share, browse, and discover plants with each other through listings, photo uploads, and in-app messaging.

## 🚀 Features

### 👤 User
- Register and log in securely
- Create plant listings with photo uploads
- Browse and search available plant listings

### 🌱 Social
- Favorite and save plants you like
- Real-time messaging between users
- Discover plants shared by the community
